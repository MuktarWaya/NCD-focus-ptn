# ระบบติดตามพฤติกรรมสุขภาพ มุ่งเป้าปัตตานี (NCDs Focus App)

แอปพลิเคชันระบบสารสนเทศสุขภาพแบบ Single-Page Application (SPA) ที่พัฒนาขึ้นด้วย Vanilla HTML, CSS และ JavaScript สำหรับติดตามและวิเคราะห์ความคืบหน้าของพฤติกรรมสุขภาพของกลุ่มเป้าหมาย ~200 คน (กลุ่มเสี่ยงและกลุ่มป่วย) ในรอบการตรวจทุกๆ 3 เดือน (Month 0, Month 3, Month 6, Month 9)

ระบบนี้ใช้ **Google Sheets** เป็นฐานข้อมูลกลางหลังบ้าน (Backend) ผ่านตัวกลาง **Google Apps Script (GAS)** และเชื่อมโยงไปแสดงผลในรูปแบบแดชบอร์ดสุขภาพที่สวยงาม สามารถใช้งานได้ง่าย ปลอดภัย และ Deploy ไวผ่าน **Netlify**

---

## ฟีเจอร์หลักของระบบ (Key Features)

1. **แผงควบคุมหลัก (Dashboard)**:
   - แสดงสถิติและอัตราส่วนกลุ่มเสี่ยงและกลุ่มป่วย
   - คำนวณอัตราการพัฒนาสุขภาพดีขึ้นแบบเรียลไทม์ (ค่าน้ำตาลลด, BMI ลด, ความดันดีขึ้น)
   - มีระบบตรวจจับสัญญาณชีพและระดับน้ำตาลวิกฤต (Urgent Alert) ของผู้ป่วย
   - กราฟสรุปผลความร่วมมือและสัดส่วนพฤติกรรมหลักในชุมชน (การออกกำลังกาย, การนอน, การทานอาหารจัด)
2. **ระบบค้นหาและกรองข้อมูลกลุ่มเป้าหมาย**:
   - ค้นหารายชื่อ บ้านเลขที่ หรือหมู่บ้านได้รวดเร็วทันใจ
   - กรองตามประเภทประชากร (กลุ่มเสี่ยง/กลุ่มป่วย)
   - กรองแยกย่อยตามโรคหลัก (โรคเบาหวาน DM, ความดัน HT, โรคอ้วน BMI เกิน, ไขมันในเลือดสูง)
3. **โปรไฟล์สุขภาพรายบุคคลเชิงลึก (Individual Profile)**:
   - แสดงข้อมูลส่วนบุคคล ข้อมูลโรคประจำตัว และยาที่ใช้ประจำ
   - ไทม์ไลน์เปรียบเทียบข้อมูลผลการตรวจวัดในรอบ 3 เดือน (M0, M3, M6, M9) พร้อมระบบเปลี่ยนสีสัญลักษณ์เตือนความเสี่ยง (เขียว/ส้ม/แดง) อัตโนมัติ
   - กราฟเส้นแสดงแนวโน้มน้ำหนัก/BMI ค่าน้ำตาล DTX ความดัน และมวลไขมัน/กล้ามเนื้อ
4. **ระบบบันทึกพฤติกรรมสุขภาพประจำวัน (Daily Log Review)**:
   - ตารางตรวจสอบคะแนนการหลีกเลี่ยงความหวาน ความมัน ความเค็ม (1-5 คะแนน)
   - บันทึกการดื่มน้ำ ระยะเวลาการออกกำลังกาย และการนอนหลับในแต่ละสัปดาห์
5. **ฟอร์มบันทึกข้อมูลสุขภาพรอบใหม่**:
   - ฟอร์มตรวจสุขภาพประจำไตรมาส ( auto-calculates BMI ในเครื่อง)
   - ฟอร์มส่งบันทึกรายวัน
   - ระบบเพิ่ม/แก้ไขรายชื่อและข้อมูลส่วนตัวกลุ่มเป้าหมาย

## ขั้นตอนการติดตั้งใช้งานจริง

ทำตามคู่มือใน [backend/README_SETUP.md](file:///g:/My%20Drive/%E0%B8%AA%E0%B8%98%E0%B8%99.%E0%B8%A1%E0%B8%B2%E0%B8%A2%E0%B8%AD/%E0%B8%81%E0%B8%A5%E0%B8%B8%E0%B9%88%E0%B8%A1%E0%B8%87%E0%B8%B2%E0%B8%99%E0%B8%9E%E0%B8%B1%E0%B8%92%E0%B8%99%E0%B8%B2%E0%B8%9A%E0%B8%A3%E0%B8%B4%E0%B8%81%E0%B8%B2%E0%B8%A3%E0%B9%81%E0%B8%A5%E0%B8%B0%E0%B8%A7%E0%B8%B4%E0%B8%8A%E0%B8%B2%E0%B8%81%E0%B8%B2%E0%B8%A3/webapp/%E0%B8%A3%E0%B8%B0%E0%B8%9A%E0%B8%9A%E0%B8%95%E0%B8%B4%E0%B8%94%E0%B8%95%E0%B8%B2%E0%B8%A1%E0%B8%9E%E0%B8%A4%E0%B8%95%E0%B8%B4%E0%B8%81%E0%B8%A3%E0%B8%A3%E0%B8%A1%20%E0%B8%A1%E0%B8%B8%E0%B9%88%E0%B8%87%E0%B9%80%E0%B8%9B%E0%B9%89%E0%B8%B2%E0%B8%9B%E0%B8%B1%E0%B8%95%E0%B8%95%E0%B8%B2%E0%B8%99%E0%B8%B5/backend/README_SETUP.md) เพื่อเซ็ตอัป Google Sheets และสร้าง Google Apps Script Web App ให้เรียบร้อย จากนั้นทำตามขั้นตอนการ Deploy เว็บไซต์ดังนี้:

### การทดสอบระบบในเครื่อง (Local Testing)
1. เปิดด้วย local web server เพื่อให้เบราว์เซอร์โหลดไฟล์ CSV ใน Demo Mode ได้ถูกต้อง เช่นรันคำสั่ง `python -m http.server 8787 --bind 127.0.0.1` แล้วเปิด `http://127.0.0.1:8787/index.html`
2. เข้าระบบด้วยรหัส `009941` ซึ่งต้องตรงกับค่า `API_PASSCODE` ของ Google Apps Script
3. ไปที่แถบ **"ตั้งค่า API"** นำลิงก์ Web App URL ที่คุณ Deploy สำเร็จจาก Google Apps Script มาวางและกรอกรหัสผ่าน API เพื่อสลับหน้าเว็บเข้าเชื่อมต่อระบบฐานข้อมูลจริงบนชีต

### การ Deploy ขึ้น Netlify
1. เชื่อมต่อโปรเจกต์นี้เข้ากับบัญชี GitHub ของคุณ ([MuktarWaya/NCD-focus-ptn](https://github.com/MuktarWaya/NCD-focus-ptn))
2. ล็อกอินเข้าสู่เว็บ Netlify ([https://www.netlify.com](https://www.netlify.com))
3. กดปุ่ม **"Add new site" -> "Import an existing project"** 
4. เลือกเชื่อมต่อกับ GitHub และเลือก Repository `NCD-focus-ptn`
5. ในขั้นตอนตั้งค่าการ Build (Build Settings):
   - **Base directory**: เว้นว่างไว้
   - **Build command**: เว้นว่างไว้ (เนื่องจากเป็น Static HTML/CSS/JS บริสุทธิ์ ไม่มีขั้นตอนการ Build)
   - **Publish directory**: กรอกจุด `.` หรือปล่อยเป็นค่าว่าง
6. กดปุ่ม **"Deploy site"** เพื่อรันหน้าเว็บจริง
7. หลังจากเปิดหน้าเว็บสำเร็จ คุณสามารถนำลิงก์ GAS Web App URL ไปใส่ในหน้าตั้งค่าภายในแอป หรือกำหนดรหัสผ่านเพื่อพร้อมให้ทุกคนในทีมกรอกข้อมูลบันทึกแบบเรียลไทม์ได้จากทุกที่ทุกเวลา!
